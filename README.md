# MTPsys
#### 体能测评系统
这个系统主要功能是对用户从excel导入的数据进行处理，根据不同士兵的年龄性别军种进行不同标准的测评，
#### 主要功能
+ 支持excel格式数据导入（按找指定格式），用户可以对部队记录进行增删查改，与此队伍相关联的所有士兵数据都会改变
+ 用户通过excel导入一整个队伍的数据，一键测评所有人的成绩，并得出此班级的合格率
+ 用户可以导出单个士兵的成绩，也可以导出整个部队的成绩，同时还可以打印所有队伍的总体成绩
